## 3. Visual Flows (For Video/GIFs)

### Flow A: The "Contain Chaos" Opening
1.  **Start**: Dark screen with subtle grid lines (`opacity: 0.1`).
2.  **Action**: Text scrambles into view: "Human By Design".
3.  **Hero**: The ID Card falls onto the screen with a heavy thud (physics animation).
4.  **Interaction**: Cursor drifts over the card; holographic sheen follows the mouse.

### Flow B: Adaptive Summaries (The "Persona Toggle")
1.  **Context**: A locked Case Study screen.
2.  **Action**: User clicks "ELI5" tab.
3.  **Result**: Text morphs into simple, friendly language. ("Big spreadsheets are hard...")
4.  **Action**: User clicks "Technical" tab.
5.  **Result**: Text rewrites itself into system architecture jargon. ("React Context API...")
6.  **Highlight**: Show the seamless transition without page reload.

### Flow C: The Decryption Lens (Hand Tracking)
1.  **Context**: "Access Denied" overlay on a project.
2.  **Action**: User activates "Gesture Mode".
3.  **Visual**: A green "Targeting Reticle" appears, tracking the user's actual hand via webcam.
4.  **Interaction**: User performs a "Pinch" gesture over the lock.
5.  **Payoff**: The lock shatters/dissolves, revealing the content.

### Flow D: The "Void" Switch (Theme Toggle)
1.  **Start**: Light Mode (Clean, industrial, crisp).
2.  **Action**: Click the Sun/Moon toggle.
3.  **Result**: The screen inverts. The background becomes deep black (`#0a0a0a`). Grid lines glow faintly. The ID card becomes a dark security badge.
