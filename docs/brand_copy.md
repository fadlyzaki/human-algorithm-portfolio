## Brand Identity

### Brand Essence
**"Human By Design"** — Every design decision filters through one question: *does this respect the human on the other end?* The brand exists at the intersection of **software logic** and **human intuition**. It's not about being clever — it's about being useful when it matters most.

### Brand Personality

| Trait | Expression |
|-------|-----------|
| Precise | Monospace typography, grid layouts, systematic spacing |
| Warm | Serif italic for emotion, amber accent for humanity |
| Resilient | Systems-thinking language, "hold it gently when it breaks" |
| Transparent | Open source code, visible system metrics, exposed processes |
| Playful restraint | Scramble text effects, emoji interest selectors, interactive tooltips — but never gratuitous |

### Brand Voice
- **Tone**: Calm authority. Think: a senior engineer who also reads philosophy.
- **Register**: Professional but human. Never corporate, never casual.
- **First person**: Always "I build" / "I design" — never "we" (solo practitioner).
- **Signature patterns**: Em-dashes for asides. Short sentences for impact. Questions as provocations.
- **Avoid**: Jargon without context, superlatives ("best", "revolutionary"), passive voice.

#### Color Philosophy
- **Amber is the soul.** It appears on philosophy borders, CTA shadows, and highlights — always signaling the "human" layer.
- **Blue is the brain.** Used for technical elements, focus areas, and system architecture.
- **Dark mode is the default personality.** The brand reads best in dark mode — ink-on-void, like a terminal.

### Typography

| Role | Family | Style | Usage |
|------|--------|-------|-------|
| **Primary** | System mono (`font-mono`) | Uppercase, tracked wide | Headlines, labels, navigation, metadata |
| **Accent** | System serif (`font-serif`) | Italic, lowercase | Subtitles ("Systems Thinker"), quotes, philosophical text |
| **Body** | System sans (`font-light`) | Regular weight, relaxed leading | Paragraphs, descriptions, long-form content |

#### Typography Rules
- Headlines always **monospace uppercase** — communicates precision and structure.
- Emotional/philosophical text always **serif italic lowercase** — communicates warmth and reflection.
- Body text always **light-weight sans** — maximizes readability, reduces visual weight.
- Font sizes scale responsively: `text-3xl → text-6xl` for hero, `text-sm` for metadata.

## Brand Do's & Don'ts

### ✅ Do
- Use monospace for anything structural or systematic
- Use serif italic for anything emotional or philosophical
- Default to dark mode for brand screenshots
- Show the "cap" emoji 🧢 beside the brand name
- Pair amber with blue for warmth + precision

### ❌ Don't
- Use rounded/bubbly fonts — they clash with the precision aesthetic
- Use bright saturated backgrounds — the brand lives in subtle, dark tones
- Use stock photography — all imagery should be personal or generated
- Say "we" — the brand is a solo practitioner
- Over-animate — every motion must have purpose (scramble = chaos, tooltip = context)
