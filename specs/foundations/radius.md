# Border Radius Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--radius-none` | `0px` | Cards, containers (sharp edge aesthetic) |
| `--radius-sm` | `2px` | Badges, small interactive elements |
| `--radius-md` | `4px` | Buttons, inputs |
| `--radius-lg` | `8px` | Modals, popovers |
| `--radius-xl` | `12px` | Large cards |
| `--radius-full` | `9999px` | Pills, avatars, status dots |

## Design Note
The system aesthetic is **primarily sharp-edged** (`--radius-none`).
Rounded elements are used sparingly for status indicators and avatars.
