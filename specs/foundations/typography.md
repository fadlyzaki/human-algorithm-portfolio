# Typography Tokens

## Font Families

| Token | Stack | Usage |
|-------|-------|-------|
| `--font-sans` | Inter, system-ui | Body text, UI labels |
| `--font-mono` | JetBrains Mono, Fira Code | Code, system labels, badges |
| `--font-display` | Inter Tight, system-ui | Large headings, hero text |

## Size Scale

| Token | Value | Usage |
|-------|-------|-------|
| `--text-xs` | `0.625rem` (10px) | Mono labels, badges, captions |
| `--text-sm` | `0.75rem` (12px) | Small body, metadata |
| `--text-base` | `0.875rem` (14px) | Default body text |
| `--text-md` | `1rem` (16px) | Emphasized body |
| `--text-lg` | `1.25rem` (20px) | Subheadings |
| `--text-xl` | `1.5rem` (24px) | Section headings |
| `--text-2xl` | `2rem` (32px) | Page headings |
| `--text-3xl` | `2.5rem` (40px) | Large display |
| `--text-4xl` | `3rem` (48px) | Hero secondary |
| `--text-5xl` | `3.75rem` (60px) | Hero primary |
| `--text-6xl` | `4.5rem` (72px) | Display hero |

## Weights

| Token | Value | Usage |
|-------|-------|-------|
| `--weight-normal` | 400 | Body text |
| `--weight-medium` | 500 | Subtle emphasis |
| `--weight-semibold` | 600 | Subheadings |
| `--weight-bold` | 700 | Headings, CTAs |
| `--weight-black` | 900 | Display numbers |

## Line Heights

| Token | Value | Usage |
|-------|-------|-------|
| `--leading-tight` | 0.9 | Display/hero headings |
| `--leading-snug` | 1.1 | Tight headings |
| `--leading-normal` | 1.5 | Body text |
| `--leading-relaxed` | 1.625 | Long-form reading |

## Letter Spacing

| Token | Value | Usage |
|-------|-------|-------|
| `--tracking-tight` | -0.05em | Display headings |
| `--tracking-normal` | 0em | Body text |
| `--tracking-wide` | 0.05em | Subheadings |
| `--tracking-widest` | 0.1em | Mono labels, badges |
