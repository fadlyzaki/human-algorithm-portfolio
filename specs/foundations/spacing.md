# Spacing Tokens

## Scale (4px base unit)

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| `--space-1` | `4px` | `p-1` | Inline padding, icon gaps |
| `--space-2` | `8px` | `p-2` | Tight component padding |
| `--space-3` | `12px` | `p-3` | Input padding |
| `--space-4` | `16px` | `p-4` | Default component padding |
| `--space-5` | `20px` | `p-5` | Card internal spacing |
| `--space-6` | `24px` | `p-6` | Section gaps, gutters |
| `--space-8` | `32px` | `p-8` | Large section padding |
| `--space-10` | `40px` | `p-10` | Section margins |
| `--space-12` | `48px` | `p-12` | Page margins |
| `--space-16` | `64px` | `p-16` | Hero section spacing |

## Rules
- Use multiples of 4px for all spacing
- Grid gutters: `--space-6` (24px / `gap-6`)
- Page margins: `--space-12` (48px / `px-12`)
- Max content width: `1280px` (`max-w-7xl`)
